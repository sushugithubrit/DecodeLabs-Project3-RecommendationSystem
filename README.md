# AI Recommendation System

## Overview

This project implements a simple recommendation system that suggests courses based on user interests and preferences.

The system uses preference matching and recommendation logic to provide relevant suggestions.

## Features

* User interest input
* Preference matching
* Course recommendation generation
* Recommendation logic implementation
* Interactive console-based interface

## Technologies Used

* Python

## Project Objective

To understand the fundamentals of recommendation systems, user preference analysis, pattern matching, and recommendation logic.
